What is CSS?


  1. CSS Stand for Cascading Style Sheets
  2. CSS Describes How HTML Elements are to be displayed on screen, Paper,or in the media.
  3. CSS is not neccesary saves a lot of work. It can Control the layout of multiple web pages all at once
  4. External stylesheet are stored in css files.

CSS can be added to HTML documents in 3 ways:

  • Inline - by using the style attribute inside HTML elements
  • Internal - by using a style element in the head section
  • External - by using a link element to link to an external CSS file.
  • The most common way to add CSS, is to keep the style in external CSS files However, this Tutorial we will use inline and internal Styles, because this is easier to demonstrate, and esier for you to try it yourself.

When comparing CSS Class vs ID, the Difference is that CSS class applies a style to multiple elements,on the other hand, applies a style to one unique element. ID is also special in that you can use A Special URL to link directaly to an element and it's used by JavaScript.

in CSS, slectors are used to target a specific element or range of elements on a web page.Once an elements has been targeted, a style or set of style can be applied to the element.

There is a wide range of selectors available. Two of the most commonly used are class and ID both are used to target elements to which a style should be applied
